1) Pull the code from this url 'https://github.com/piyushagrawal28/VehicleInsuranceTest.git'.
2) Open the above code in your IDE using Open Project option.
3) Make sure your IDE has Maven and TestNG plugins.
4) Test data is present in '.\VehicleInsuranceTest\src\test\resources\testdata\InsuranceFormData.xlsx' file.
5) Don't change the structure of the excel, you can update the value in the excel as per your need.
6) If you want to add new sheet in the excel then follow the same naming convention like InsuranceFormData1, InsuranceFormData2 and so on.
7) Make sure new sheet also have same structure of row and coloumn as per the existing sheet.
8) Test-cases present in '.\VehicleInsuranceTest\src\test\java\carInsuranceRegistration\CarInsuranceTest.java' file.
9) Open the CarInsuranceTest.java file in the IDE.
10) Right click on the file and click on Run As -> TestNg Test to run the testcase.
11) Result will be present in test-output folder.
